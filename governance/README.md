### Advanced Techniques Demonstrated
These prompts incorporate:
- **Personas**: "You are a [role] with [specific objective]"
- **Chain-of-thought**: "Think through this systematically..." with step-by-step reasoning
- **Structured output**: Specific formats, tables, and organization schemas
- **Constraint-based**: "You must only use information provided..." to prevent hallucination
- **Few-shot learning**: Examples embedded to guide output quality
- **Negative prompting**: Explicit "Do NOT..." instructions
